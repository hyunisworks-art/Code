# ストリートファイター6 コンボ入力ツール

ストリートファイター6向けの **コンボ入力テキスト作成支援ツール**です。
方向入力・ボタン入力をクリック操作で組み立て、
コンボ表記をテキストとして整理・保存できます。
本ツールは **個人利用を目的** としています。

---

## 概要

本ツールは、以下を目的として作成されています。

* コンボ入力表記を手入力せず、操作的に作成する
* 矢印表記／数字表記の切り替えに対応する
* クラシック操作・モダン操作の両方に対応する
* コンボと補足情報（ダメージ・メモ）をまとめて管理する

すべて **Webブラウザ上で完結** し、ローカル環境でも動作します。

---

## 主な機能

* 表示モード切替（矢印表記／数字表記）
* 操作モード切替（クラシック／モダン）
* 方向キー・ボタン入力のクリック入力
* 技の区切り入力（→）
* 特殊条件入力（例：壁限定、遅らせ など）
* コンボパーツの登録・呼び出し
* ダメージ数値の入力・保存
* メモ入力（コピー時に別行で出力）
* コンボの保存・一覧表示
* 一覧でのフィルター・ソート機能

---

## 基本的な使い方

1. ボタン操作でコンボを組み立てます
2. 出力欄に生成されたコンボが表示されます
3. ダメージやメモを入力できます
4. 保存するとコンボ一覧に追加されます
5. コピー操作で、コンボとメモをまとめて取得できます

---

## 対応環境

* デスクトップ：Chrome / Firefox / Edge / Safari（最新版）
* モバイル：iOS / Android のモダンブラウザ

※ JavaScript が有効である必要があります
※ FileReader API / LocalStorage / Clipboard API を使用しています
